# ERP-to-Warehouse Data Sync Automation

## Overview
Windows-based automation app to sync data between ERP and warehouse systems with monitoring and alerting.

## Problem
Manual monitoring and sync failures caused delays and lack of visibility.

## Solution
Built a C# application that:
- runs sync automatically
- tracks status
- logs activity
- sends failure alerts

## Tech Stack
C# | .NET | SQL Server | JSON | SMTP

## Features
- Automated sync execution
- Real-time status tracking
- Failure alert emails
- Logging system

## Impact
- Reduced manual effort
- Improved reliability
- Faster issue detection

## Architecture
ERP → DataSyncApp → Warehouse
